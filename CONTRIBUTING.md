# Contributing

Thanks for contributing to Agent Foundry.

## What Helps Most

- bug reports with a real repository shape or stack example
- feature requests tied to a concrete workflow gap
- generated-file examples that feel too generic or incorrect
- screenshots or short recordings of confusing UX

## Local Workflow

1. Install dependencies with `npm install`.
2. Run `npm run verify`.
3. Run `npm test`.
4. Run `npm run package` if your change affects packaging or manifest behavior.
5. Use the Extension Development Host from `.vscode/launch.json` when changing the user experience.

## Product Direction

Agent Foundry is not trying to replace coding agents. It should make repositories work better with them.

Bias contributions toward:

- better repo analysis
- better generated workflow assets
- better handoff quality
- better support for common repository shapes
