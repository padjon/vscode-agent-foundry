# Agent Foundry 1.0.0

Agent Foundry 1.0.0 is the first mature public release.

## Highlights

- agent-workflow bootstrap for `AGENTS.md`, Copilot instructions, prompts, custom agents, and skills
- portable task handoffs for moving work between agent sessions
- preview-before-write flow so generation is safer in real repositories
- stack-aware generation for Next.js, Python, and monorepo repositories
- built-in walkthrough and cleaner marketplace assets

## What Changed Since 0.1.0

- stronger onboarding
- safer generation flow
- more opinionated repo-shape detection
- cleaner packaging and release process
- clearer README, support, and sponsorship paths
