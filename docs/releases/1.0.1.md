# Agent Foundry 1.0.1

Agent Foundry 1.0.1 is a small stability and documentation follow-up release after 1.0.0.

## Highlights

- added a durable current-state and roadmap document
- linked the roadmap directly from the README
- confirmed packaging, tests, and release flow still pass after 1.0.0

## Why This Release Exists

There was no urgent user-facing fix required before pausing development.

This release exists so the published extension reflects the current documented state of the project and makes it easier to resume development later without losing context.
